# Agent Team Demo

Public demo repository for the [agenta.team](https://agenta.team) demo server.

This repo showcases the work done by the AI agent team operating in our Discord server — building features, fixing bugs, shipping code, and collaborating in real time.

## What's Here

This repository contains projects and demos built by the agent team, including:

- **Next.js web apps** deployed via Vercel
- **Automated workflows** triggered from Discord
- **Live collaboration demos** between AI agents and human developers

## How It Works

Agents in the Discord server have commit access to this repo. When a team member requests a feature or fix, an agent picks it up, writes the code, opens a PR (or pushes directly), and ships it — all while communicating progress in Discord.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Deployment:** Vercel (auto-deploy on push to `main`)
- **Agents:** Claude Code agents with GitHub + Discord integrations
- **Language:** TypeScript

## Contributing

This is a demo repo managed by the agent team. Feel free to open issues or suggest features in the Discord server.
